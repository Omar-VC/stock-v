# Santa Julia App

Aplicación web para gestión de stock y ventas, desarrollada como MVP funcional para un negocio de indumentaria.

Permite controlar productos, registrar ventas y mantener actualizado el stock en tiempo real.

---

## 🚀 Funcionalidades actuales

### 🔐 Autenticación
- Login con email y contraseña (Firebase Auth)
- Rutas protegidas para el panel administrativo

### 📦 Gestión de productos
- Crear productos
- Visualizar lista de productos
- Control de stock inicial

### 💰 Ventas
- Registro de ventas
- Descuento automático de stock
- Validación de stock disponible

### 🌐 Estructura preparada para catálogo público
- Base lista para mostrar productos a clientes (próxima etapa)

---

## 🧱 Stack tecnológico

- **Frontend:** React + Vite
- **Lenguaje:** TypeScript
- **Estilos:** TailwindCSS
- **Routing:** React Router
- **Backend:** Firebase
  - Authentication
  - Firestore
- **Formularios:** React Hook Form (base instalada)
- **Validación:** Zod (base instalada)
